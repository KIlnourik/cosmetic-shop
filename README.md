# Личный проект магазина косметики Maroon
SPA приложение написанное на Typescript, с помощью библиотеки React, с использованием Redux. Сборка на Vite

## Что реализовано:
Онлайн магазин косметики Maroon состоит из трех страниц: 
1. Главная страница
2. Страница каталога
3. Страница продукта

## Что применял:
1. React
2. Redux
3. Redux Toolkit
4. Axios
5. Vite
